URL Shortener Microservice

You can pass this API an URL and it will respond with a shortened link to access that URL.
Note: the URL you provide must start with ftp://, http:// or https://

To shorten an URL:
https://atomk-fcc-urlshort.glitch.me/new/https://www.example.com

Sample output:
{ "url-original": "https://www.example.com", "url-short": "https://atomk-fcc-urlshort.glitch.me/xmpl" }

To access the shortened URL:
https://atomk-fcc-urlshort.glitch.me/xmpl

Sample error output:
{"error":"This route requires a parameter. Use /new/:url instead."}
