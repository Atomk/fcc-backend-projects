Timestamp Microservice

Pass timestamps in milliseconds! Not seconds, or you'll get wrong dates! If the conversion gives unexpected results try to add three zeros at the end of your timestamp.

Example usage:
https://atomk-fcc-timestamp.glitch.me/December%2015,%202015
https://atomk-fcc-timestamp.glitch.me/1450137600000

Example output:
{ "unix": 1450137600, "natural": "December 15, 2015" }
